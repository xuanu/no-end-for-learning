
# AutoJs记录   
> 官网： https://pro.autojs.org/    
> 简单介绍下：通过无障碍服务来操作手机，实现自动化操作。    

##### 辅助工具   
1. [WEditor](https://github.com/alibaba/web-editor) ; 可以用于获取控件信息   
2. [Uiautomator2](https://github.com/openatx/uiautomator2);是一个Python下的自动化工具，可以用于操作手机。
